Landslide Risk in Kutawaringin

In 2021, I created a landslide disaster mitigation WebGIS, which consists of a landslide hazard map and an evacuation route map to the shelter. It is hoped that this WebGIS can be used by the community and local sub-district officials to analyze the level of landslide hazard in their area; thus it can be used as a decision-making analysis to create an area that is ready to coexist with disasters. I produced a functional web map application with a customized map palette in codeigniter framework (equipped with HTML/CSS & AdminLTE template) and display GIS data on the web with Leaflet.js library. Through my considerations of functionality and design, this project allowed me to better understand project methodologies and technical requirements associated with webGIS development.
